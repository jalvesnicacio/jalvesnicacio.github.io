Social Learning System

Dekstra: Um ambiente de Aprendizagem Social para Iniciação à Aprendizagem de Programação Usando Esquemas de Concepção e Planos de Programação (2018)

This paper proposes the design and construction of a Virtual Environment Learning Program, based on the concept of Social Learning Environment. In the proposed Environment, a set of supports was provided at the interface to assist the learner in developing the algorithms building with the Blockly library.
