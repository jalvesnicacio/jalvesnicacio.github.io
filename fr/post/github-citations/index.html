Soutien pour les citations issues de dépôts GitHub.

Apprenez à rendre disponibles des fichiers de citation au style APA ou BibTeX depuis vos dépôts.

Jalves Nicacio
Oct 5, 2021 2 min de lecture research

Comment faciliter la citation de votre travail par les autres ?

Dans le travail académique, nous développons souvent des prototypes ou rendons des visualisations de données disponibles via des dépôts. Une manière de rendre ces artefacts de recherche disponibles est à travers la plateforme GitHub, qui nous permet non seulement de rendre publics des logiciels, prototypes, ou données, mais aussi de contrôler leur versionnement.

Cependant, lorsque nous écrivons un article et souhaitons référencer un dépôt GitHub, cela peut être un peu difficile de savoir comment mieux référencer ces dépôts. Avec cela à l’esprit, Github a récemment créé un moyen de soutenir officiellement les citations vers le dépôt de données ou de code. Cette nouvelle fonctionnalité, développée par GitHub, permet aux chercheurs et universitaires d’informer la communauté et les parties intéressées sur comment citer correctement leurs dépôts, spécialement dans les articles scientifiques.

Pour ce faire, les chercheurs doivent ajouter un nouveau fichier à leur dépôt : CITATION.cff. Selon Arfon Smith, un membre de l’équipe Github, les informations dans ce fichier sont utilisées pour créer des liens de citation pratiques au style APA ou BibTeX qui peuvent être référencés par d’autres.

Pour activer cette fonctionnalité dans votre dépôt, vous pouvez simplement créer un fichier nommé CITATION.cff dans votre dépôt. Utilisez l’exemple ci-dessous comme modèle pour votre fichier :

cff-version: 1.2.0
message: "Si vous utilisez ce logiciel, veuillez le citer comme ci-dessous."
authors:
  - family-names: Druskat
    given-names: Stephan
    orcid: https://orcid.org/0000-0003-4925-7248
title: "Mon Logiciel de Recherche"
version: 2.0.4
doi: 10.5281/zenodo.1234
date-released: 2021-08-11

Vous pouvez également utiliser le site cffinit pour remplir les informations de votre fichier de citation. Vous pouvez obtenir plus d’informations sur le format de fichier cff sur cette page : What is a CITATION.cff file?
